## 📦 Installation

```bash
poetry add git+https://github.com/tobiasdanielangelo/my_django_app.git
```

⚙️ Setup

1. Add to INSTALLED_APPS:

```
# settings.py

INSTALLED_APPS = [
...
'my_django_app',
]

```

2. Run migrations (if any):

```
python manage.py migrate
```

3. Use it in your project!

🧪 Running Tests

```
python manage.py test my_django_app
```
